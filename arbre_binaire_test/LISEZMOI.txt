Arbre Binaire test
==========================================

Ce programme teste les fonctionnalités de la librairie "arbre_binaire".

Compiler le programme
---------------------

  - Tout compiler:
		
***

	make

***

	- Pour les autres types de compilation (Debuggage seulement, statique seulement, etc.), voir le fichier Makefile.

Éditeur
-------

  - Le fichier de projet Code::blocks est disponible dans ce répertoire. Seulement ouvrir le fichier "arbre_binaire_test.cbp" dans cet éditeur.
